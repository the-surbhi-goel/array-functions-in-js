Push

It'll push the value at the top of the array

CHECK CONSOLE FOR OUTPUT



Existing push

    const arr1 = [1, 2, 3, 4];
    arr1.push(5);


Custom push

    Array.prototype.myPush = function (value) {
        let len = this.length;
        this[len] = value;
    };
    const arr2 = [1, 2, 3, 4];
    arr2.myPush(5);
